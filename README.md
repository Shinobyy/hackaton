# Stack technique

- Front : React avec Vite comme bundler
- Back : Express.js avec BDD PostgreSQL

# Installation

```bash

cd front && npm i
cd back && npm i

```

# Lancer le front

```bash

cd front && npm run dev


```

# Lancer le back

```bash

cd back && npm run dev

```
